# ADS_2016
Qui dentro è caricato il mio progetto per l'esame di Advanced Data Science 2026 che contiene file rds dei libri analizzati, file markdown per procedere con l' analisi, il relativo file html e il ppt di presentazione per l'esame. 
Inoltre ho caricato eventuali file tidy.rds nel caso ci dovessero essere problemi vari- 
Prima di tutto bisogna scaricare i vari file rds:
- murakami_english.rds nel quale sono presenti tutti i 15 romanzi di Haruki Murakami scritti in inglese
- others_books.rds che servirà successivamente e nel quale sono presenti i romanzi di 4 autori molto famosi che hanno influenzato molto l'autore (1984, The Great Gatsby, The Long Goodbye, Trout Fishing in America)
Successivamente si può procedere con il file markdown ed eseguire i vari comandi.
